# Token

Java Project

Project centered around a hypothetical in which physical Tokens 
are given out with various functionalities such as Natural Disaster Alerts,
a digital currency wallet with the ability to purchase itmes and accept donations,
recieve and keep a uniquie ID for the Token, and the ability to track the Token
in the case of a missing person or other non-specific reason by way of splitting
the town or surrounding area of the Token holder into Quadrants.

The code in this project is built to enable said functionalities.
